# Cartridge Images

This directory contains cartridge images for different Pokémon games. Each image should be a 60x60px PNG file that represents the game's cartridge artwork.

## Required Images:
- yellow.png - Pokémon Yellow
- blue.png - Pokémon Blue
- red.png - Pokémon Red  
- green.png - Pokémon Green
- crystal.png - Pokémon Crystal
- emerald.png - Pokémon Emerald
- firered.png - Pokémon FireRed
- heartgold.png - Pokémon HeartGold
- platinum.png - Pokémon Platinum
- black.png - Pokémon Black
- white.png - Pokémon White

## Image Specifications:
- Format: PNG
- Size: 60x60 pixels
- Should show the game's cartridge artwork/logo
- Transparent background preferred
